# Portrade_FrontEnd

Portfolio와 Trade의 합성어로 다양한 분야의 포트폴리오를 업로드 할 수 있는 공간입니다.

## 핵심 기능

-   사용자 관리
    -   회원가입 및 로그인, 로그아웃
    -   sessionStorage 를 사용해 유저별 token 발급 및 로그인 유지
-   기업 및 공고 등록
    -   기업 등록 후 기업별 공고 등록, 조회수 및 마이페이지에 저장
-   CRUD
    -   1:1문의, 공지, 자주 묻는 질문 CRUD 구현
-   carousel & modal
    -   기업 공고 carousel과 , 공고 상세 modal 간 라우팅 연결
